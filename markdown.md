# ¿Qué es un archivo .md?

Un archivo .md (Markdown) es un tipo de archivo que permite escribir texto con formato de manera sencilla.

Se usa mucho en plataformas como GitHub para documentación.

## ¿Cómo previsualizar Markdown?

En editores como Visual Studio Code puedes ver cómo se verá tu archivo con:

`ctrl + shift + v`

## Sintaxis básica de Markdown

### Encabezados

```js
# Título 1
## Título 2
### Título 3
```

### Texto

```js
*Itálica* o _Itálica_
**Negrita**
***Negrita e itálica***
~~Texto tachado~~
```

### Lista enumerada

```js
1. Primero
2. Segundo
3. Tercero
```

### Lista con viñetas

```js
- Elemento 1
- Elemento 2
  - Subelemento
```

### Enlaces

```css
[Texto del enlace](https://ejemplo.com)
```

### Imágenes

```css
![Texto alternativo](https://url-de-imagen.com)
```

### Código

````css
Linea de código: `console.log("Hola mundo");`

Bloque de código:
```js
console.log("Hola mundo");

```
````

### Tablas

```css
| Nombre | Edad |
|--------|------|
| Juan   | 25   |
| Ana    | 30   |
```

### Líneas horizontales

```css
---
```
