# ACTIVIDAD: ¿Qué he aprendido de Gt?

## Objetivo

Con el fin de aprender el flujo de git, cada alumno pondrá en un segmento de texto acerca de su persona, hobbies, habilidades que ha aprendido recientemente, cosas que hizo durante vacaciones, etc.

## Instrucciones

1. Clonar el repositorio

   ```md
   git clone URL_DEL_REPOSITORIO
   cd NOMBRE_DEL_REPO
   ```

2. Antes de trabajar, actualizar:

   ```
    git pull
   ```

3. Editar el archivo ejercicio.md

   ```
   Agregar tu sección al final del documento:
    ## TuNombre
    - Me gusta...
    - En vacaciones hice...
    - Algo que me pareció interesante fue...
   ```

4. Guardar cambios:

   ```
   git add .
   git commit -m "nombre de la persona"
   ```

5. Subir cambios
   ```
   git push
   ```

## Importante

Si tienes error al hacer push, ejecuta:

```
git pull
git push
```

## Nota

Antes de subir cambios, asegúrate de tener la versión más reciente del proyecto.
